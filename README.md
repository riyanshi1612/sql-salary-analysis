
Data Science Salary Analysis — SQL Project

SQL • SQLite • DB Browser • Data Analysis

## Overview

A SQL-based analysis of data science job salaries across multiple countries, job titles, and employment types. Built using SQLite and DB Browser for SQLite. This project is a companion to the Excel Salary Dashboard project — same dataset, analyzed using SQL queries.

## Queries Covered

✅ Query 1: View all data

✅ Query 2: Average salary by job title

✅ Query 3: Top 5 highest paying jobs

✅ Query 4: Count of jobs by country

✅ Query 5: Average salary by country

✅ Query 6: Jobs where salary is above average (Subquery)

✅ Query 7: Count of jobs by job schedule type

✅ Query 8: Average salary by job schedule type

✅ Query 9: Top hiring platform by job count

✅ Query 10: Full-time jobs in United States with salary above $100K

## SQL Concepts Used

SELECT, WHERE, GROUP BY, ORDER BY, LIMIT, AVG, COUNT, ROUND, Subqueries, Aggregate Functions, Filtering with AND

## Database Schema

Table: jobs — job_id, job_title, salary_year_avg, job_country, job_schedule_type, job_platform
## Key Insights

📌 Senior Data Scientists earn the highest average salary (~$154K/year)

📌 United States has the highest number of data science job postings

📌 Full-time roles offer significantly higher salaries than Part-time or Contractor roles

📌 Indeed is the top hiring platform by job count

## Tools Used

SQLite, DB Browser for SQLite
